Куйдін Владислав ПД-13
1)Git — розподілена система керування версіями файлів та спільної роботи. 
2)GIT - має 3 основні стани:1)commited  2)modified  3)staged
3)add - для того ,щоб додати певний файл в репозиторій
4)status - для перевірки статусу git
5)commit - фіксування змін у файлах
6)init - запуск певного репозиторію
7)потрібні для запису назви коміту
8) git log - список комітів
9)checkout для копіювання файлів з історії або сцени в робочу директорію
